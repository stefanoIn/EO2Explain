What This Form Expects

  • Single-band Sentinel-2 L2A GeoTIFFs, one file per band.
  • The uploaded bands should refer to the same area of interest and the same before/after event pair.
  • The rasters should already be cropped to the event area and readable by the EO pipeline as standard TIFF files.

Band Usage In This Project

  • Flood workflow: B03 and B08 only.
  • Wildfire workflow: B04, B08, and B12.
  • The wildfire pipeline computes NBR and dNBR with B12.
